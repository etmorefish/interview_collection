# 50. HTTPS 是什么？HTTPS ，实际就是在 TCP 层与 HTTP 层之间加入了 SSL/TLS 来为上层的安全保驾护航，主要用到对称加密、非对称加密、证书，等技术进行客户端与服务器的数据加密传输，最终达到保证整个通信的安全性。

[![img](https://image-1302243118.cos.ap-beijing.myqcloud.com/img/a9e8780d6b899cf5709579fc9c451fd9)](http://static.iocoder.cn/a9e8780d6b899cf5709579fc9c451fd9)

一句话概括：**HTTP + 加密 + 认证 + 完整性保护 = HTTPS** 。
